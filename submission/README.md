# 최종 제출 자료

- `제안서_최종본.txt`: 팀 합의문 기준 원문 (`제안서_최종본.docx`와 항상 같은 내용이어야 한다)
- `제안서_최종본.docx`: 온라인 제안서 업로드용 원본 (Word 작성, 디자인 요소 없는 순수 텍스트형)
- `제안서_최종본.pdf`: 위 DOCX를 Word에서 그대로 내보낸 제출용 PDF
- `온라인_제출용_3단계_요약_최종.txt`: 온라인 신청서 입력용

공개 사이트용 PDF 사본은 `../assets/final-proposal.pdf`, 찾기 쉬운 루트 사본은 `../최종_제안서.pdf`다. 세 PDF는 항상 `제안서_최종본.pdf`와 같은 파일이어야 한다.

이전에 사용하던 디자인 버전(2페이지 그래픽 레이아웃) PDF와 생성 스크립트는 `../archive/drafts/`, `../archive/build-workbench/proposal-source-styled/`로 옮겼다. 팀 피드백에 따라 디자인 요소 없는 순수 문서형으로 전환했기 때문이다.
