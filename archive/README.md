# 보관 자료

- `drafts/`: 제안서 초안·시안 (`초기초안` → `중간수정본` → 시안 A/B → `STYLED_superseded` 순으로 진행됐다가, 팀 피드백으로 디자인 요소 없는 순수 문서형으로 최종 전환)
- `supplementary/`: 기존 보완자료 이미지
- `build-workbench/proposal-source-styled/`: 폐기된 STYLED 버전을 생성한 최종 빌드 스크립트
- `reference/`: 공모전 안내 원본
- `git-bundles/video-history.bundle`: 원격 저장소가 없던 기존 영상 저장소의 전체 Git 이력

이 디렉터리는 현재 공개 사이트에서 직접 사용하지 않지만, 제작 과정과 복구 가능성을 위해 보존한다. 제안서 생성 과정에서 나온 중간 디버그 로그·폰트 점검 파일 등 재사용 가치가 없는 스크래치 파일은 보존하지 않았다.
